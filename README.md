# Fitness Tracker App

Desktop aplikacija za praćenje prehrane izrađena u Pythonu koristeći Tkinter i Pandas biblioteku.

## Funkcionalnosti

- unos hrane i gramaže
- automatski izračun kalorija
- izračun proteina, ugljikohidrata i masti
- dnevni pregled unosa
- spremanje izvještaja u CSV format
- učitavanje spremljenih izvještaja
- grafičko korisničko sučelje (GUI)

## Korištene tehnologije

- Python
- Tkinter
- Pandas
- Matplotlib
- CSV baza podataka
- GitHub
